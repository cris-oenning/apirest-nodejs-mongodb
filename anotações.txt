API REST com Node.js e MongoDB

- A API é baseada no framework express para tratar as requisições e enviar as respostas aos endpoints;
- Foi utilizado o MongoDB como banco de dados, porém com a ODM Mongoose;
- Foi realizado as operações no MongoDB Atlas;
- O pacote Nodemon ajudou a atualizar o código em tempo real da aplicação;
- A API foi testada pelo software Postman;

Objetivo do projeto
- Projeto de Estudo de uma criação de uma API nos padrões RESTful, que possui um CRUD;
- Os endpoints foram criados com os verbos HTTP que correspondem a ação do mesmo;
- As respostas foram baseadas em JSON, retornando também o status correto;
- Foi aplicado validações simples, para simular o 'mundo real';
